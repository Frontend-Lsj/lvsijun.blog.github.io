Express介绍#

  • Express 官方网站：https://expressjs.com/

  • Express 官方网站（中文）：https://expressjs.com/zh-cn/

Express 安装#

安装 express：

1

npm install express -g

安装 express-generator，安装之后，可使用应用程序生成器工具 (express) 快速创建应用程序框架。

1

npm install express-generator -g

查看安装的 express 版本：

1

express --version
