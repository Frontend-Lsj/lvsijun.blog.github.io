08-Vuex

Vue2
1 min · 2020/01/01

  • 2、实际应用

state / mutations / getters / actions => 公共数据库 / 直接改变数据方法 / state的计算属性 / 异步( 调用mutations方法 )间接改变数据

1、store.js文件#
let store = new Vuex.Store({
  state: {
    tableHeight:0,
    tableSHeight:0,
    tableTHeight:0,
  },
  getters: {
    getCommonHeight: (state) => (typeKey) => {
      return state.tableHeight + "px"
    },
  },
  mutations: {
    changeTableHeight(state,payload){
      state.tableHeight = payload.height;
      state.tableSHeight = payload.secondH;
      state.tableTHeight = payload.thirdH;
    }
  },
  actions: {
  	changeTableHeightSetTime(ctx){
       setTimeout(()=>{
           ctx.commit('changeTableHeight', {height:100,secondH:200,thirdH:300});
       },2000)  
    },
  }
})

export default store
2、实际应用#
this.$store.state.tableHeight;
this.$store.commit('changeTableHeight', {
          height:460,
          secondH:390,
          thirdH:490
})
this.$store.dispatch('changeTableHeightSetTime');
this.$store.getters.getCommonHeight;


// 从vuex中按需导入mapState\mapMutations\mapActions\mapGetters函数
import { mapState,mapMutations,mapActions,mapGetters } from 'vuex'
// 通过刚才导入的函数,将当前组件需要的全局数据,映射为当前组件的computed计算属性
computed: {
    ...mapState({'tableHeight'})
    ...mapMutations(['changeTableHeight']) //this.changeTableHeight()
    ...mapActions(['changeTableHeightSetTime']) //this.changeTableHeightSetTime()
    ...mapGetters(['getCommonHeight'])
}
